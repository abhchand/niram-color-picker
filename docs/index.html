This wonderful blog post from RefactoringUI outlines a main problem with most modern color palette pickers:

You can't build anything with five hex codes. To build something real, you need a much more comprehensive set of colors to choose from.

Inspired by the methodology of the above blog post, Niram Color Picker is a tool to generate a full palette of colors for your next project.

Made by @abhchand. Report any feedback or issues on github.

Niram is the word for "color" in my native language.
